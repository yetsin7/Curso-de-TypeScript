# Practica guiada — Async y fetch

## Objetivo

Trabajar con promesas y datos externos sin perder tipos.

## Actividades

1. Crea una funcion `async` que retorne un dato tipado.
2. Tipa la respuesta esperada de una API.
3. Maneja error con `try/catch`.
4. Explica por que los datos externos deben validarse aunque uses TypeScript.
5. Ejecuta el ejemplo compilado si el modulo ya lo permite.

## Mini reto

Construye una funcion que consulte una lista de posts y devuelva solo id y titulo con tipos correctos.
