# Practica guiada — Utility types

## Objetivo

Reducir repeticion y modelar variaciones de datos con herramientas nativas de TypeScript.

## Actividades

1. Usa `Partial` para actualizar un objeto.
2. Usa `Pick` para extraer un subconjunto de propiedades.
3. Usa `Omit` para quitar campos sensibles.
4. Usa `Record` para mapear categorias a valores.
5. Explica donde cada utility type te ahorra codigo real.

## Mini reto

Modela una API de cursos donde exista un tipo completo, uno resumido y uno editable.
