# Practica guiada — Tsconfig y tooling

## Objetivo

Entender como la configuracion cambia el nivel de seguridad del proyecto.

## Actividades

1. Lee `tsconfig.json` y explica `strict`.
2. Identifica para que sirve `outDir`.
3. Ejecuta `npm run check`.
4. Ejecuta `npm run build`.
5. Explica por que no conviene desactivar reglas solo por comodidad.

## Mini reto

Propone una configuracion minima para un proyecto pequeno y justifica cada opcion principal.
