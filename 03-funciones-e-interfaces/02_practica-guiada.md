# Practica guiada — Funciones e interfaces

## Objetivo

Describir contratos claros para datos y comportamiento.

## Actividades

1. Crea una funcion con parametros tipados.
2. Crea una funcion con valor de retorno explicito.
3. Define una interfaz para usuario o curso.
4. Usa esa interfaz como parametro de una funcion.
5. Prueba una firma invalida y corrige el error.

## Mini reto

Construye una funcion que reciba una leccion tipada y devuelva un resumen listo para mostrar en consola.
