# Practica guiada — Introduccion

## Objetivo

Entender que problema resuelve TypeScript y como se relaciona con JavaScript.

## Actividades

1. Cambia el mensaje del ejemplo principal.
2. Provoca un error de tipo intencional y lee el mensaje del compilador.
3. Ejecuta `npm run check`.
4. Ejecuta `npm run build`.
5. Compara el archivo `.ts` con el `.js` generado en `dist/`.

## Mini reto

Explica con tus palabras por que TypeScript ayuda en desarrollo, pero no reemplaza a JavaScript en ejecucion.
