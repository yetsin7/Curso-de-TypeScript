# Practica guiada — Proyecto final

## Objetivo

Integrar tipos, modulos y ejecucion en un mini proyecto mas real.

## Actividades

1. Recorre `src/models.ts` y explica cada tipo.
2. Recorre `src/lesson-manager.ts` y explica su responsabilidad.
3. Agrega una propiedad nueva al modelo de leccion.
4. Ajusta los datos de ejemplo para cumplir el nuevo contrato.
5. Ejecuta `npm run lesson:project`.

## Mini reto

Agrega una funcion para listar solo las lecciones pendientes o solo las completadas.
