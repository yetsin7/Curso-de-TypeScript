# Documentacion oficial de TypeScript

TypeScript se aprende mucho mejor cuando acompanias el curso con el handbook y
la referencia oficial del lenguaje.

## Referencias principales

- [TypeScript documentation](https://www.typescriptlang.org/docs/)
- [TypeScript handbook](https://www.typescriptlang.org/docs/handbook/intro.html)
- [TSConfig reference](https://www.typescriptlang.org/tsconfig/)

## Recursos clave para estudiar

- [Tipos basicos](https://www.typescriptlang.org/docs/handbook/2/basic-types.html)
- [Narrowing](https://www.typescriptlang.org/docs/handbook/2/narrowing.html)
- [Funciones](https://www.typescriptlang.org/docs/handbook/2/functions.html)
- [Objetos y interfaces](https://www.typescriptlang.org/docs/handbook/2/objects.html)
- [Generics](https://www.typescriptlang.org/docs/handbook/2/generics.html)
- [Classes](https://www.typescriptlang.org/docs/handbook/2/classes.html)

## Orden recomendado

1. Refuerza tipos basicos y anotaciones.
2. Practica funciones, objetos e interfaces.
3. Pasa a unions, narrowing y generics.
4. Luego estudia configuracion, modulos y proyectos reales.
